# MDXセットアップ完了

これはMDXのサンプル記事です。MDXを使うことで、Markdownの中にReactコンポーネントを埋め込むことができます。

## MDXの特徴

- **Markdown + JSX**: MarkdownとJSXを組み合わせて使える
- **コンポーネント**: カスタムReactコンポーネントを記事内で使用可能
- **動的コンテンツ**: インタラクティブな要素を記事に追加できる

## コード例

```javascript
const greeting = "Hello, MDX!";
console.log(greeting);
```

## まとめ

MDXを使うことで、より豊かな表現力を持つブログ記事を作成できます。
